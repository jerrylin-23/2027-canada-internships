## 🛠️ How it Works
This repository uses a zero-token scraper script ([crawl.js](./crawl.js)) that hits Greenhouse, Lever, Ashby, and SmartRecruiters APIs directly for **150+ North American employers**.

### Run locally
```bash
npm install
node crawl.js
```
